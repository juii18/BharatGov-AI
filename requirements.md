## Project Overview
GovGuide is an AI-powered platform that helps citizens discover government schemes based on their eligibility, occupation, and personal situation. The system analyzes user input and recommends relevant schemes from a structured government scheme dataset.

## Problem Statement
Many citizens are unaware of government schemes they are eligible for due to lack of proper guidance and centralized access.

## Solution
GovGuide provides an intelligent interface where users can enter their occupation or situation, and the system recommends relevant schemes using Natural Language Processing and data matching techniques.

## Objectives
- Help citizens discover relevant government schemes
- Simplify scheme search process
- Improve accessibility and awareness
- Provide personalized recommendations

## Features
- User-friendly interface using Streamlit
- Scheme recommendation based on eligibility
- Fast search using structured dataset
- Scalable and deployable architecture

## Target Users
- Farmers
- Students
- Workers
- Entrepreneurs
- General citizens

## Technology Stack
- Frontend: Streamlit, HTML, CSS
- Backend: Python
- AI/NLP: Text processing and matching
- Database: Government Scheme Dataset (CSV)
- Tools: GitHub, VS Code

## Expected Outcome
Users can quickly discover schemes relevant to them, improving access to government benefits.
